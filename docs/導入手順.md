# 導入手順

## 1. 導入前確認

| 項目 | 確認内容 |
|---|---|
| Windows | Windows 11を推奨 |
| 権限 | 可能なら管理者権限で実行 |
| Python | 会社のソフトウェア導入許可リストに従って導入済みであること |
| Node.js | 必要な場合のみ、会社のソフトウェア導入許可リストに従って導入 |
| Codex / Claude Code | 標準インストーラーまたは会社指定手順で導入 |

## 2. インストール

```powershell
cd <展開先>
.\installer\install_standard.ps1 -ConfigureClaude -ConfigureCodex
```

ユーザーPATHに `ai-pip` / `ai-npm` を追加する場合：

```powershell
.\installer\install_standard.ps1 -ConfigureClaude -ConfigureCodex -AddWrappersToUserPath
```

## 3. 導入後確認

```powershell
.\installer\check_status.ps1
```

## 4. ライブラリ導入確認

許可済みの例：

```powershell
ai-pip install pandas
```

review扱い・未許可の場合は停止され、AIガバナンスチームまたは管理者への相談メッセージが表示されます。

## 5. セルフ・セキュリティ・チェック

ローカルリポジトリに仕組みができたら、組織で定めたセルフ・セキュリティ・チェック手順に従って確認してください。

本パッケージではセルフ・セキュリティ・チェック本体は同梱しません。既存の社内ツールや別リポジトリを利用する場合は、そのREADME/手順を正とします。
