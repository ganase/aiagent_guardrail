# Claude Code 標準指示

## 前提

- この環境は標準インストーラーで導入され、標準設定・ガードレール・テンプレート・バージョン管理の仕組みが適用されています。
- ガードレールは完全ではありません。外部情報、承認操作、機密情報、ライブラリ導入は慎重に扱ってください。

## 必須ルール

- `.env`、秘密鍵、APIキー、接続文字列、個人情報、本番データを読ませたり、コードや社内Gitリポジトリへ含めたりしないでください。
- 未承認MCP、未承認Plugin、個人契約API、未承認SaaSに接続しないでください。
- Python / Node.js / Git などのランタイム本体導入は、会社のソフトウェア導入許可リスト・指定手順に従ってください。
- ライブラリ導入は `ai-pip` / `ai-npm` または標準Hookで許可済みパッケージリストを確認してから実行してください。
- 許可リスト外、review扱い、または判断できないライブラリは導入せず、AIガバナンスチームまたは管理者へ相談してください。
- 危険コマンド、外部送信、ファイル大量削除、権限変更、費用発生を伴う処理は、実行前に目的・対象・影響を説明し、人間の承認を求めてください。
- ローカルリポジトリに仕組みができたら、セルフ・セキュリティ・チェックを実行してください。
- 仕組みを継続利用・チーム共有・部内業務利用に進める場合は、READMEを整備し、社内Gitリポジトリにコミット・プッシュしてください。
