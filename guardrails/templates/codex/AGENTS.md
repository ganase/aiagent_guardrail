# AI Agent 標準指示

## 前提

このリポジトリでは、AI Agent標準ガードレールを前提に作業します。

## ランタイム・ライブラリ導入

- Python / Node.js / Git などのランタイム本体導入は、会社のソフトウェア導入許可リスト・指定手順に従います。
- pip / npm のライブラリ導入は、標準インストーラーで配布された許可済みパッケージリストに従います。
- 可能な場合は `ai-pip` / `ai-npm` を使って導入可否を確認してください。
- 許可リスト外、review扱い、または判断できないライブラリは導入せず、AIガバナンスチームまたは管理者に相談してください。

## 安全確認

- `.env`、秘密鍵、APIキー、接続文字列、個人情報、本番データを扱わないでください。
- 危険コマンド、外部送信、削除・上書き、大量変更、権限変更の前には必ず承認を求めてください。
- ローカルリポジトリに仕組みができたら、セルフ・セキュリティ・チェックを実行してください。
- 継続利用・共有・業務利用に進む場合は、READMEを整備し、社内Gitリポジトリにコミット・プッシュしてください。
